# 🏨 Hospeda+

Sistema de gerenciamento hoteleiro desenvolvido em Java para a disciplina de Programação Orientada a Objetos.

## 📖 Sobre o Projeto

O **Hospeda+** é um sistema de gerenciamento de hotéis que permite controlar hóspedes, funcionários, quartos, reservas, serviços adicionais e registros demonstrativos de pagamento.

O projeto foi desenvolvido com foco na aplicação prática dos conceitos de Programação Orientada a Objetos.

## 🎯 Objetivos

- Aplicar conceitos avançados de POO;
- Utilizar herança e polimorfismo;
- Implementar interfaces;
- Trabalhar com coleções Java;
- Utilizar tratamento de exceções personalizadas;
- Desenvolver um sistema funcional e organizado.

## 🚀 Funcionalidades

### 👤 Gestão de Pessoas

- Cadastro de hóspedes;
- Cadastro de funcionários;
- Consulta de informações cadastrais.

### 🛏️ Gerenciamento de Quartos

- Quarto Simples;
- Quarto Duplo;
- Suíte;
- Controle de disponibilidade;
- Controle de ocupação.

### 📅 Sistema de Reservas

- Criação de reservas;
- Confirmação de reservas;
- Cancelamento de reservas;
- Associação de hóspedes e quartos;
- Cálculo do valor total da hospedagem.

### 🛎️ Serviços Adicionais

- Café da manhã;
- Lavanderia;
- Serviço de quarto;
- Transporte.

### 💳 Controle de Pagamentos

- Registro demonstrativo de pagamento;
- Associação de pagamento à reserva;
- Consulta das informações financeiras.
